# Accounts module

This module owns master business accounts.

A business account is the central container controlled by a master owner. One account can contain many businesses, shops, workers, roles, and permissions.

## Files

- `mod.rs` exports the accounts module files.
- `model.rs` defines the business account model.
- `repository.rs` will contain database queries for account records.
- `service.rs` will contain account creation and ownership rules.
